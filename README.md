# node-ts

Simple Node and typeScript app

## Quick Start

1. `npm install`
2. `npm run serve-debug`

## Included

- basic `tsconfig.json`
- npm scripts to help build, run, watch, and debug
- concurrently, ts-node, ts-lint, typescript for helping with builds
- types for node with @types/node
